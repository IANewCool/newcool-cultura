# NewCool Cultura - Patrimonio Chile

Portal educativo sobre cultura, museos y patrimonio en Chile.

## Demo

[https://cultura.newcool.io](https://cultura.newcool.io)

## Caracteristicas

- Directorio de centros culturales
- Buscador de museos y bibliotecas
- Informacion de fondos concursables
- Patrimonio cultural de Chile
- Eventos culturales

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
